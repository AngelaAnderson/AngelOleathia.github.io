ANGEL OLEATHIA

Central

Role: UX Designer/Researcher

As the UX researcher and designer for this project, I was tasked with researching and identifying a target market that would benefit from a new cloud storage solution, develop a strong branding strategy, and ultimately bring a new, unique cloud storage solution to market. Considering the workload commonalities amongst creatives, this solution extends beyond basic cloud storage services to include customer relationship management (CRM) capabilities.

Problem

Despite the vast amount of competition, the cloud storage market is still young and viable for new competitors given the right combination of features. This desired solution needed to leverage a “network effect,” by way of collaboration. More specifically, it needed to allow users to save content from the web, organize content using filters (such as categories, tags, groups etc.), create content, and upload files.

Solution

The solution is Central. Central is a cloud storage and organization workspace that provides professional creatives with a convenient solution to managing projects and working relationships.

Competitive Analysis

The project kicked off by conducting a competitive analysis of key market competitors consisting of: Google Drive, Drop Box, and Evernote. While each of these competitors have numerous strengths that are advantageous for their brand, an analysis of their weaknesses presented an opportunity to create unique features specific to this new cloud storage solution. Some of the unique opportunities posed from the competitive analysis included: uploading and streaming music and video files, advanced organization, use of filters and tags, uploading file in the background, and Sorting files by date.

User Survey

After conducting market research, I conducted a user survey which aided in developing user personas. In total, 18 respondents completed the survey, providing a reasonable amount of data to analyze. Almost 95% of respondents reported that they currently use or have you used Cloud Storage to store data. The 12 respondents stated they utilize Google Drive, 7 respondents indicated they use iCloud, 5 indicated they use DropBox, and another 5 stated they use Microsoft One Drive. The majority of respondents indicated they use cloud storage almost every day for personal use, however, about 24% said they mainly use cloud storage for work. This data not only confirmed that cloud storage services are in high demand, but it also identified a niche market of professionals who could potentially benefit from this solution.

User Stories

I want to upload individual files and folders

I want to be able to organize my files by tags

I want to save items directly from the web

I want to be able to make client profiles

I want to be able to collaborate with others

Branding

Central inspires productivity and motivates creativity as a luxury online workspace and data storage solution. The brand itself aims to motivate creatives to produce high quality work and inspire artistic confidence which is communicated in the selected primary and seconday colors.

Typography

While pondering the ideal typography for this project, I knew I wanted to incorporate thin stokes into the design to keep the overall aesthetic simplistic and metaphorically connect back to the theme of creativity. I chose ParalucentStencil Light as my primary font as it communicates the theme of the brand well. However, the font is too overpowering to be used on text heavy pages. Thus, I paired it with Monsterrat Light to be used for headings and Acumin Pro Regular to be used for body text. All three fonts paired well, eliciting a modern and professional appearance.

Conclusion

The research done in this project to create this solution has taught me a lot, enforcing the idea of designing with the user in mind. I’ve also have a better understanding of how brand messaging influences an over design. Initially, I tried incorporating illustrations into the design but found that it took away from a chic-professional appeal. I found it better to include photographs and geometric shapes to communicate professionalism while still inspiring creativity. Given more time, I would have thought out a plan to grow the collaboration and network feature. However, I find the CRM aspect of the solution a helpful tool that would be of value to professional creatives.
