Gabriel B. Taye Foundation

  • ANGEL OLEATHIA

Role: Web Designer

Client: Gabriel B. Taye Foundation

Summary

This project was one of the first websites that I designed. I hadn’t yet enrolled into Bloc’s frontend coding bootcamp and I had no knowledge of the standard UX desgin proccess. Nevertheless, I felt inclined to include this project in my portfolio because it was the most meaningful project I’ve had the pleasure to work on.

Problem

Gabriel Taye was an 8-year-old student who sadly committed suicide as a result of constantly being bullied in school. I was asked by his mother to design this website to supplement her anti-bulling efforts.

Conclusion

BusyBus was the first project I completed as part of Bloc’s coding bootcamp. From this assignment, I learned how to utilize user research in a meaning way to create an everyday solution for communicating public transit information to users. After reciving feedback from prototype testing and helpful tips from Bloc’s grading team, I also picked up some helpful tips on how to keep a desing simple and asetheticly appealing while communicating necessary information.
